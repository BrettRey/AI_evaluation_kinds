# AI evaluation kinds

Working repository for a framework paper applying Boyd's homeostatic property cluster theory and Khalidi's causal-network kind theory to the vocabulary of AI evaluation and ML governance.

**Working thesis:** AI evaluation vocabulary contains several kinds of kind-term. *Hallucination* and *jailbreak* are HPC- or Khalidi-network candidates with identifiable sustaining mechanisms. *Reasoning*, *agency*, and *emergence* are capability-attribution terms whose projectibility depends on benchmark design and deployment context. *Alignment* and *trustworthiness* are umbrella governance constructs that organise lower-level kinds. The framework distinguishes these and explains why a single treatment of "what is X?" misfires.

**Status:** seedling. Working material in [`NOTES.md`](NOTES.md).
